# Paladar Sur - Landing Page

Premium landing page for Paladar Sur, featuring a minimalist design, Apple-inspired aesthetics, and high-conversion elements.

## Features
- **Modern Design**: Built with vanilla HTML, CSS, and JS.
- **Responsive**: Fully optimized for all device sizes.
- **Interactive**: Smooth transitions and animations.

## Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/designwebrich-debug/PaladarSur.git
   ```
2. Open `index.html` in your browser.

## Project Structure
- `index.html`: Main structure.
- `styles.css`: Custom styling and design system.
- `script.js`: Interactive logic.
- `assets/`: Image and brand assets.
